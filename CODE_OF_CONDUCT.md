# Code Of Conduct

This project follows the Contributor Covenant Code of Conduct.

## Our Standards

- Be respectful and direct.
- Assume good intent, but be specific about technical concerns.
- Focus criticism on ideas, code, and behavior, not people.
- Avoid harassment, discrimination, and personal attacks.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, and other contributions that violate these standards.

## Scope

This applies within project spaces and in public spaces when someone is representing the project.

## Reporting

Report conduct issues to `support@stately.ai`.

For the full Contributor Covenant text, see [contributor-covenant.org](https://www.contributor-covenant.org/).
